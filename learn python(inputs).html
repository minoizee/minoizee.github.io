Learn how to add inputs in Python








Last page I said I will walk you through how to maket the program interactive and have user inputs. This is very simple and combines variables and a new kind of print statement; the input. All you need to do to extend your variable is type: 




name=input("What is your name?")


Like with print statements you can put whatever you want between the apostrophes and likewise with the variable name. The variable on the other side of the "=" is just like every other variable and you can put it in a normal print statement so that people can see what they typed in. Remember, you always have to make an input statement with the variable.

Remember to not put quotation marks in the brakets, otherwise it will display the exact contents within the quotes rather than what is inside the variable.
